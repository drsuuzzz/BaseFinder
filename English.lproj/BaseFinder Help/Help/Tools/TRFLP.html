Terminal Restriction Fragment Length Polymorphism (T-RFLP)
 

The T-RFLP tool is intended to aid in the analysis of T-RFLP experiments. Namely, it calculates the peak positions according to the size standard. It also calculates the intensity of each peak which is then saved in an external file as peak areas. The identified peaks are redrawn with a bold line accompanied by the peak location in the data view window.

This tool performs best after Fitted Baseline Adjust (if not done by the automated DNA sequencing software) Matrixing (if not done by the automated DNA sequencing software) and smoothing--Filter Convolution, Filter-FFT or Deconvolution. Please see the quick start guide for some pointers on determining if your data needs to be processed by the above mentioned tools. In addition, the peak positions may be identified incorrectly due to mobility of the fluorescent dyes. The Mobility Shift: Polynomial can be used to correct for the drift in the channels.

To use the tool, specify the low peak cutoff values for channels 1 and 2. These values indicate that every peak above this value is a valid peak. Every peak below this value is invalid.

Another option to set is the Cleanup Intense Peaks option. Sometimes the detector in the DNA sequencer can become overloaded and a peak will seem very tall and flat or with a strange dip in the center. Some machines perform automatic color separation (see Matrixing) as the data is being collected. Matrixing of these intense peaks leaves some interesting artifacts behind and may also cause the dip in the center of the peak. This option will attempt to correct for this and cleanup the artifacts and "unfold" the peak.

Select the appropriate Size Standard to use for your data. At this time only one standard is supported with peaks indicating the following base locations: 25, 30, 40, 50, 75, 100, 125, 150, 200, 250, 300, 350, 400, 450, 475, 500, 550, 600, 650, 700, 750, 800, 850, 900, 950 and 1000.

Lastly, select the location and enter a name for the output file that will store the measured peak intensities/areas by pressing the Save button. After the tool has completed its execution, the file will record for each channel (0 and 1), for each identified peak, the position in the trace, the interpolated base, the area of the peak, and the relative area. The area for all peaks is totaled at the end.
