Filter FFT
 

The raw data coming off most automated sequencing hardware has both high and low frequency noise which makes basecalling of peaks difficult. This tool uses the fast fourier transform (fft) to implement a special profile bandpass filter to remove this high and low frequency noise.

Filtering is usually best done after the data has been matrixed and before the data is basecalled.

The Low Cutoff and High Cutoff values specify the range of frequencies which will remain after filtering. Frequencies below Low Cutoff and frequencies above High Cutoff will be removed. Take care not to set Low Cutoff too high or High Cutoff too low otherwise you could filter out true dye-primer peaks. These cutoff values should only need to be reset if the sampling frequency of the automated sequencer or the run conditions of the gel has been changed.

The filter has a side-effect of creating negative dips around dye-primer peaks. This results in a raised baseline. One way to remove these negative values is by setting Remove Values Below (%). All values that have values below this point are set to zero. Too high a value will tend to remove small peaks, while too small a value doesn't remove all negative values. Another option is to use the Histogram Cutoff Tool.

By default the filter is applied to all channels, but one can limit the filter to only specific channels.
