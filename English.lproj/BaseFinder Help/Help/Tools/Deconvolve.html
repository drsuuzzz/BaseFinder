Deconvolution
 

The electrophoretic separation used in DNA sequencing introduces an effect called zone broadening. This effect reduces interpeak resolution by widening the fragment populations during separation. The zone-broadening process is generally equivalent in effect to convolution with a Gaussian-based function. The reduced resolution attributable to band spreading causes reduced accuracy in base-calling. The effect becomes more prominent as the electrophoresis time increases, thereby contributing to the difficulty of accurately base labeling the later portions of data collected from a sequencing separation. The deconvolution process partially reverses the effects of band spreading, thereby increasing resolution and base-calling accuracy. The process is sensitive to noise, and if overapplied can result in the appearance of false peaks, so it must be used with care.

The Deconvoution tool attempts to estimate the spread function as a function of time. In addition, the algorithm attempts to estimate a data set that when convolved with the spread function results in the actual observed data set. The estimated data set is the end result of the algorithm, displayed to the user.

The details of the algorithm are described in Giddings et al, 1998. For the purposes of interpreting the parameters in the tool it should be noted that the Number Iterations parameter indicates the number of iterations or repetitions the algorithm goes through to estimate the data set. The parameter alpha influences the calculations such that negative or excessively large values are "avoided" in the estimations of the data set (Giddings et al, 1998).

The zone broadening function is approximated by a symmetrical Gaussian. The width of the Gaussian peak, sigma, varies peak by peak and thus the sigma function is estimated by a polynomial: a + bx + cx^2. The default estimates for a, b, and c are displayed when first selecting the tool. Selecting the New button provides options for calculating new estimates for the a, b, and c coefficients. The user can either manually enter the coefficients estimated by other tools or allow the coefficients to be estimated using the gaussPeaks file created by the Gauss Peak Fitting tool. This file contains a representative sample list of position (time) versus sigma values.
