Scale Factor
 

It is often the case that different dyes will exhibit different intensity ranges in the same trace. This makes it difficult to visualize all the channels on a single scale. The Scale Factor Tool can be used to modify the scale of individual channels.

The Scale Factor Tool will rescale the data, to increase peak size or decrease peak size. The tool will multiply the data with a user specified scaling factor. Each channel can have a separate scaling facter applied. Values between 0 and 1 will decrease peak magnitude. Values greater then 1 will magnify peak magnitude. A scale factor of 1 is used for channels that will not be scaled.
