Manual Deletion
 

Often there is little usable data at the beginning and end of a data file. Also when the primer peak comes through, it often exceeds the sensitivity limits of the detection equipment. All these regions should be deleted to improve the appearance of the data, and to improve basecalling accuracy.

Range When this option is selected, one can highlight the region of data to be deleted or zeroed. This is accomplished by clicking in the data view window and dragging the mouse. Dragging off the window causes the view to scroll. The user can also manually enter the range.

If the Leave space switch is set, deleting a data region that is internal to the data set will result in the "deleted" data points being set to zero, rather than literally deleted.

By selecting or deselecting channels, it is possible to limit the region set to zero to the selected channels. If the Leave space switch is not set then the region is deleted in all channels.

From End When this option is selected, the user must enter the number of points to delete from the end of the data. These points will then be deleted.

Although the data is deleted, the numbering of the data points shown in the data view window will remain unchanged.
