Fitted Baseline Adjust
 

The Fitted Baseline Adjust tool is a tool to remove background signal and normalize the baseline. The user specifies a window size which should be larger then the width of a peak. If the window is smaller then a peak width, then signal data may be erroneously removed. The tool searches for the minimum signal intensity within each window of the user specified size. It performs a linear interpolation between the minima identified in each window. And then subtracts the line formed by connecting each minima. As a result, each channel shifts downward to be on the same baselined axis.

To find the size of a peak, select in the data view window each side of the peak and subtract. Position information is shown at the top of the window.

This is typically the first tool executed after loading the data.
