Primer Peak Deletion
 

The Primer Peak Deletion tool calculates and deletes data before the primer peak in order to reduce the quantity of data that is to be processed by other tools. Data after the full length extension peak is also deleted.

The tool looks for the largest peak in the data by looking in either all the channels or in a single channel. Select Precalculate primer peak to see where the algorithm will delete data.

Although the data is deleted, the numbering of the data points shown in the data view window will remain unchanged.
