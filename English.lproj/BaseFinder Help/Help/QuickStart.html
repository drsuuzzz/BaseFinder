This is a guide to getting started with BaseFinder. A walkthrough of loading data and selecting tools is presented.

1. Open a data file. Select tools to process the data by selecting them from the tool inspector.

2. If your trace looks like this,

the baseline has drifted and needs to be adjusted before using any of the tools. To correct for baseline drift, use the Fitted Baseline Adjust tool. This is typically the first tool run. After running this tool the drift will be corrected and the data will look like this.

3. The next step is to perform a color separation. This is necessary due to spectral overlap of the fluorescent dyes used. The Matrixing tool adjusts the data set such that each channel represents an independent color source. This tool may not be necessary if color separation/matrixing was performed by the DNA sequencing equipment.

4. Typically, the next step in the process is to smooth the data using one of the smoothing tools to remove jitter from the trace, Filter-Convolution, Deconvolution, or Filter-FFT.

5. With fluorescently labeled DNA sequencing, the different dye used in the separation alter the speed at which the attached DNA fragments travel through the capillary. cDNA fragments of the same length attached to different dyes will have different elution times and thus, channels will appear shifted from one another. The shift can be corrected by using the mobility shift tools. The following is an example to illustrate the concept using the Mobility Shift: Polynomial tool.

6. The signal tends to decay over long reads and BaseFinder provides tools to correct the signal decay. If RNA is being analyzed, the tool Signal Decay Correction should be executed at this point. The Signal Decay Correction tool corrects an observed exponential decay in the signal.

For analyzing DNA, the appropriate tool is Normalization: Signal Average. This tool will average the signal onto a common scale for consistent peak intensities.

7. For RNA analysis, individual channels can be rescaled with the Scale Factor Tool to aid visualization of the trace to overcome issues with dye quantum yields.

8. At this point the data is ready for processing by tools such as Base Calling (DNA) and Align and Integrate (SHAPE).
