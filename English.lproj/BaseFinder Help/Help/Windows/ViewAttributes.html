Display Options
 

Five panels are accessible from the Display Options menu at the bottom center of the data-file window.

Split Channels

This panel allows the user to separate the channels to view them individually in the same data file window. Split Allwill split the dispaly to view each channel separately. Split Halfwill pair the first and second channels together in the upper half of the data view window and the 3rd and 4th channels paired together in the lower halv. This is a useful option for the Align and Integrate tool. Split None does not split the channels.

Data markings

This panel allows the selection of how the data is marked in the upper portion of the data file window. Either the data point is displayed or if bases exist, the position of the base in the sequence.

In addition, long hash marks can be displayed which identify the peak positions for a base in a sequence. It is possible to disable the markings by unchecking Mark Base Location.

Channel Attributes

This panel enables the user to choose the color (or gray) in which to display each channel. It also allows the user to disable some channels altogether.

The channel labels are parsed to determine how bases should be marked in the data view and the base-sequence panel. There should be at least one one-letter word in each label. The last of these is used as the label.
Labels

This panel is used to enable and disable display of base labels in the data view.
Scaling

This panel is used to specify normalization options for display of the data.

For each scaling method, a min and a max are calculated for each channel. A data point with the minimum value for its channel will be displayed at the baseline of the dataview. Similarly, a data point with the maximum value for its channel will be displayed at the top of the view. Other values will be scaled linearly.

If all channels are normalized on a common scale, they will all have equal min and max values, which will be the minimum and maximum over all the channels. If the channels are normalized on independent scales, the minimum and maximum will be calculated separately for each channel. The final option is to allow the user to enter an arbitrary range.
