The Base Sequence Panel
 

The base-sequence panel appears only when a base-sequence exists for a given set of data. In addition to presenting the letter corresponding to each base, it also gives that base's numerical position in the sequence, its confidence, and its location in data points relative to the current data set.

In addition to displaying this information, it is also possible to save the sequence into a text file (see Saving Base Sequence).

The base-sequence panel and the data window work cooperatively to help in the display of sequence information. If the user clicks in a data view (without dragging the mouse), the closest base to that click will highlight and that same base will be highlighted in the base-sequence panel. If a base is highlighted in the base-sequence window (by clicking in the desired row), the same base will be highlighted in the data window.

The data-file window can also be resized, closed and minimized in the usual manner for windows.
