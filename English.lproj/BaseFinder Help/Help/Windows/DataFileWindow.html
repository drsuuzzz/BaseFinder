The Data File Window
 

When a data file is loaded into the program, it is plotted and displayed in the Data view window. The specifics of modifying the display are discussed under Viewing Data.

The major part of the window is taken up with the data display area. Clicking at any point in the data display will cause the values in each channel at that point to be displayed in the Data Tracking area at the top of the window. The left and right arrow keys can then be used to move the selected point. The horizontal magnification and the number of views are controlled by the two pop-up menus at the bottom right of the window. Other display options are listed in the pop-up list at the bottom center.

There are three sets of controls across the top: the Base Tracking area for viewing the number of the selected base, the Data Tracking area for viewing the point number and values at the selected point, and the Lane Selection area for choosing a lane to view. In both the Base Tracking and Data Tracking areas, turning auto track on causes the corresponding fields to update automatically as the mouse is moved around the data display.

At the bottom left of the window are a set of color boxes which indicate the color of each of the numbered channels.

The data-file window can also be resized, closed and iconified in the usual manner for windows.
