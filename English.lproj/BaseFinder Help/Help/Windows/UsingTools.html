The Tool Inspector
 

The tools inspector provides a compact interface to BaseFinder's data processing and analysis tools. The drop down menu provides a list of tools that can be selected. Making a selection from this menu will result in the appropriate tool interface being displayed in the panel. The tool can then operate on the data.

If a data file has not been loaded it will not be possible to activate any tool. Some tools will not allow the user to make selections or any changes.

The Append button initiates the execution of the tool. Select this button after making the appropriate settings in the tool.

The Replace button moves to the position formerly held by the Append button after the tool operation has completed. If selecting a brand new tool, Append will again be in the far right. Select this button if you wish to make changes to the previously run tool or to replace a tool in the script with another.

The Delete button will delete the tool from the script as well as any modifications it may have made to the loaded data.

The Insert button will insert a tool into the script. In the scripting inspector, select the position you wish to add the tool, then select Insert. The tool will be inserted above the selected tool.
