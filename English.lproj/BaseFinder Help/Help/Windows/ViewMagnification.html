Changing magnification of views
 

It is possible to change the number of views displayed in the data file window. A view is a section of the data displayed in the data file window. For instance, 2 views would show the data split into two different sections. It is ordered such that where the top one ends the bottom one begins.

To make display of the data graphs more flexible, the horizontal magnification of the views can be changed with the pop-up menu at the bottom right of the Data File Window. A magnification value of 100% corresponds to a data point width of one pixel.

Since the view can be scrolled horizontally, changing the horizontal magnification also changes the amount of the data that will be visible at any given point. In addition to numerical magnification values, the data can be displayed so that a correct magnification is calculated to fit all the data in the visible region.

In addition to horizontal magnification, it is also possible to apply a vertical magnification. To the right of the data file window, there is a sliding scale. To increase or decrease magnification, drag the scale upwards or downwards. To reset, select the reset button. The vertical scale magnification selection will apply to the printing of the data.
