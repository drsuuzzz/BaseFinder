BaseFinder Menu
 

About BaseFinder

Selecting this option presents an panel giving version information about BaseFinder. The window is dismissed by clicking the close button in the upper left corner.

Preferences

When selecting this option a new panel will be displayed which allows the user to select preferences for Expertise, Resources or Misc.

Expertise.
The level of expertise selected effects what the user is allowed to do within BaseFinder.

Beginner Level: In this mode the user is only allowed to:
  1. load scripts
  2. step through scripts seeing effect of each tool and tool's parameters
  3. apply script to multiple lanes
Novice Level: In this mode the user is allowed to do all actions of the beginner level plus the following additional actions:
  1. edit and save scripts
  2. users have a private set of scripts and tool resources
  3. can set separate execution and editing points
Expert level: In this mode the user is allowed to do all actions of the novice level with the following exceptions:
  1. single clicking on a script entry only sets the position in the script. It does not show the parameters of the current tool in the script.
  2. double clicking on a script entry will show that tools parameters in the ToolPanel.

Resources:
This directory specifies where BaseFinder can load external BaseFinder tools. These tools are additional tools which do not come as default with the BaseFinder platform and will be loaded at start up.

Misc.
Checking the debugging option will allow additional information messages to be output to the console. This information can be useful when encountering a problem. Copy/paste any console messages when reporting BaseFinder bugs.
