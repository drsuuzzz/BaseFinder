Edit Menu
 

Undo

All data processing operations can be undone. Currently only the most recent data processing step can be undone. Applying undo again will undo the undo and bring the data back to the most recent processed state.

Undo is accomplished through the edit menu, or by typing command 'z'.

Redo

The Redo edit menu item will initiate a step forward in the data processing to execute the next tool in the script. Command-shift-z will also initiate the Redo.
