Acknowledgements

BaseFinder was originally developed as a tool for processing trace data for DNA sequencing. More recently it has been enhanced to support the analysis of RNA structure.

A quick note about terminology when reading this Help file. A trace file or data file is the raw output from DNA sequencing equipment in a format as specified in Loading the Data. Lanes are individual trace files. Multiple lanes/files can be loaded at a time in BaseFinder. Channels within a trace are the individual fluorescent intensity measurements of each fluorphore used in an experiment. Click a topic to the left to learn more.

Also, see the following articles.

Giddings MC, Severin J, Westphall M, Wu J and Smith LM, "A Software System for Data Analysis in Automated DNA Sequencing", Genome Research, 1998, 8:644-665

Vasa SM, Guex N, Wilkinson KA, Weeks KM and Giddings MC, "ShapeFinder: A software system for high-throughput analysis of RNA structure", In preparation.
