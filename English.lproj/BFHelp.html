BaseFinder Help

There is not any extensive help available for BaseFinder . . . Yet.

However, there is some context help available, which can be used by pressing the help key and clicking on the associated user interface element - you can see what you get!  There should be some helpful tidbits gained in doing so - explore.

An important reminder: Without an appropriate processing script, BaseFinder will likely not do anything remotely close to what you want it to do.  You can define a script by first opening some data, selecting tools in the tool inspector, setting appropriate parameters, and inserting/appending the tool to a script.  However, it will probably take you a very long time to learn what all of the appropriate processing parameters are.  A better approach is to use one of the example scripts which should have come with the BaseFinder distribution you recieved and modify it as necessary.

Also, remember, this is beta software.  We request you report any and all bugs - that way we can fix them (though no guarantees on how quickly)!  Send bug reports, comments, suggestions, and so on to giddings@unc.edu.

Stay in touch - let us know any problems you're having, and how BaseFinder is doing for you. Also, keep an eye on the web site for the latest updates.
